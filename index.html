O que é o d.efeitos?

O d.efeitos é um site que reune efeitos CSS bastante criativos, mostrando também seu código e ensinando como fazê-lo.

Como funciona ?

Basicamente você terá um catálogo cheio de efeitos, tanto para botões, links, ícones e imagens. Basta você escolher o seu efeito preferido e clicar no link logo abaixo dele que você será redirecionado para um arquivo no Github com o código comentado do efeito em questão.

Tem algum custo ?

Não. Esse é um projeto Open Source, todos os efeitos (e seus códigos) são disponibilizados de graça para qualquer um que quiser deixar seu projeto moderno. Inclusive, quem tiver algum efeito bacana que quiser compartilhar, fique a vontade para enviar um pull request.
