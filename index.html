What is this?

The cool-effects is a website that gathers very cool CSS effects, teaching how to do it and showing its code as well. The effects were taken from sites like Codrops and CSS-Tricks.

How it works?

Basically you will have a catalog full of effects for buttons, links, icons and images. Choose your favorite effect and click on the fork icon right below it and you will be redirected to a Github file with the code of this effect.

It has any cost?

No. This is a Open Source project, all effects (and its codes) are available for free for anyone that want to make your project more beautiful and modern.
